# Install

```sh
pip install git+https://github.com/0xlakshay/sanic_kaioken
```

# Usage

Simply import, nothing else

```
import sanic_kaioken # noqa
```

> [!NOTE]  
> Sanic Kaioken respects sanic's debug flag & will only patch logging in debug mode.
